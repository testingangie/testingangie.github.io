Нужно протестировать приложение, которое обрабатывает данные о книгах, которые есть в наличии в книжном магазине.
У приложения нет UI (User Interface), приложение получает на вход запрос по REST и так же возвращает ответ.


Продумать тест-план.

Какие техники тест-дизайна?
Какие предвидите «узкие» места, которые стоит протестировать особенно тщательно?

Задание по SQL:




1. Вывести, какие книги Булгакова есть в магазине.
2. Вывести название, жанр и цену тех книг, количество которых больше 8, в отсортированном по убыванию цены виде.
3. Вывести название всех книг каждого автора, если книг некоторых авторов в данный момент нет на складе – вместо названия книги указать Null.
